⚡ DEVOPS COMMANDS

// Linux • Git • GitHub — Quick Reference

🐧 LINUX
🌿 GIT
🐙 GITHUB
// Navigation
pwd
Current folder ka path dikhao
ls
Folder ke andar ki files dikhao
ls -la
Sari files detail ke saath dikhao (hidden bhi)
cd foldername
Us folder mein jao
cd ..
Ek folder peeche jao
cd ~
Home folder mein jao
// Files & Folders
mkdir foldername
Naya folder banao
touch file.txt
Nayi khali file banao
cp file.txt copy.txt
File copy karo
mv file.txt /path/
File move ya rename karo
rm file.txt
File delete karo
rm -rf foldername
Folder aur sab kuch andar se delete karo
// File Content
cat file.txt
File ka content terminal mein dikhao
less file.txt
File scroll karke padhne ke liye kholo
head -10 file.txt
File ki pehli 10 lines dikhao
tail -10 file.txt
File ki aakhri 10 lines dikhao
echo "text" > file.txt
Text file mein likho (overwrite)
echo "text" >> file.txt
Text file ke aakhir mein add karo
grep "word" file.txt
File mein koi word dhundo
// System Info
whoami
Abhi kon sa user logged in hai
uname -a
System aur OS ki info dikhao
df -h
Disk space kitna hai dikhao
free -h
RAM kitni use ho rahi hai dikhao
top
Live CPU aur RAM usage monitor karo
ps aux
Sare running processes dikhao
kill -9 PID
Kisi process ko force stop karo
// Permissions
chmod +x script.sh
Script ko run karne ki permission do
chmod 755 file
Owner ko sab, baaki ko read/run permission
sudo command
Admin (root) ki tarah command chalao
chown user file
File ka owner badlo
// Networking
ping google.com
Internet connection check karo
curl url
URL se data download karo ya test karo
ifconfig
Network interfaces aur IP address dikhao
ssh user@ip
Remote server se connect ho jao
// Shell Scripting
#!/bin/bash
Script ki pehli line — bash script hai yeh batao
./script.sh
Script run karo current folder se
bash script.sh
Script directly bash se run karo
$1 $2 $3
Script ko diye gaye arguments read karo
$?
Aakhri command ka exit code — 0 matlab success
crontab -e
Scheduled tasks edit karo (cron jobs)
// Setup
git config --global user.name "Name"
Apna naam Git mein set karo
git config --global user.email "e@mail.com"
Apni email Git mein set karo
git config --list
Sari Git settings dikhao
// Repository
git init
Current folder ko Git repo banao
git clone URL
GitHub se project download karo
git status
Kya kya changes hain abhi dikhao
git log --oneline
Commit history short mein dikhao
// Save Changes
git add file.txt
Ek file ko commit ke liye tayyar karo
git add .
Sari changed files ko tayyar karo
git commit -m "message"
Changes save karo message ke saath
git commit -am "message"
Add aur commit ek saath karo
// Branches
git branch
Saari branches dikhao
git branch branchname
Nayi branch banao
git checkout branchname
Doosri branch pe chale jao
git checkout -b branchname
Nayi branch banao aur uspe chale jao
git merge branchname
Doosri branch ko current mein merge karo
git branch -d branchname
Branch delete karo
// Undo
git restore file.txt
File ki changes wapas karo (commit se pehle)
git reset HEAD~1
Aakhri commit undo karo (changes raho)
git stash
Changes temporarily chhupao (baad ke liye)
git stash pop
Chhupaye hue changes wapas lao
// Remote Connection
git remote add origin URL
GitHub repo se apna local repo jodo
git remote -v
Connected remote repos dikhao
git remote remove origin
Remote connection hata do
// Push & Pull
git push origin main
Apna code GitHub pe upload karo
git push -u origin main
Pehli baar push karo aur link set karo
git pull origin main
GitHub se latest code apne paas lao
git fetch
GitHub se updates check karo (merge nahi)
git push origin --delete branch
GitHub se branch delete karo
// SSH Setup
ssh-keygen -t ed25519
Nayi SSH key banao GitHub ke liye
cat ~/.ssh/id_ed25519.pub
Public key dikhao — GitHub mein add karni hai
ssh -T git@github.com
GitHub SSH connection test karo
// Workflow (Daily Use)
git pull → code → git add . → git commit -m "" → git push
Roz ka basic DevOps workflow yahi hai
git diff
Kya kya changes kiye hain last commit ke baad
git shortlog -s -n
Sab contributors aur unke commits count dikhao
